## 概要
VRCのアバター検索において、[PoligonLoading](https://polylo.power-of-tech.com/)のタグ検索をベースにレコメンドシステムを作る個人的なプロジェクトです。
## 目的
最終的な目標は、以下を可能にすることです。  
  
ユーザーが **3〜5個のタグを選択**するだけで
- **同じ嗜好を持つユーザーが実際に使用しているアバター**
- **同じ嗜好を持つユーザーに好まれているアバター**
   
をレコメンドできる仕組みを構築すること

単なるタグ一致ではなく、  
「**人の嗜好の重なり**」をベースにした推薦を目指す。
